Holiday Joke & Fact Bot

Sends a daily Christmas joke or fun fact from a curated Google Sheet. Perfect for learning database reads and filtering.

üé¨ Video coming soon

üìù Pre-work (Before Recording)

  • Create a Google Sheet with columns: days_until_xmas, type, content, source
  • Populate with 30 jokes/facts using AI (prompt below)
  • Connect your Google account to n8n
AI Prompt to Generate Content 
Generate 30 rows of Christmas content for a daily email countdown.

Format as a table with these columns:
- day_number (1-25, with days 24 and 25 having 2-3 extras each)
- content_type (joke or fact)
- content (the actual joke or fact)

Requirements:
- Mix roughly 60% jokes, 40% facts
- Family-friendly and age-appropriate for kids
- Jokes should be actually funny, not just puns
- Facts should be interesting conversation starters
- Day 25 content should feel extra special for Christmas Day

Output as a table I can copy directly into Google Sheets.

‚ö° Workflow Overview

Schedule ‚Üí Code ‚Üí Sheets ‚Üí Filter ‚Üí Gmail

This workflow runs daily, calculates the current day number, reads your content from Google Sheets, filters for today's joke or fact, and sends it via Gmail.

üìã Step-by-Step Guide

  1. 1

    Schedule Trigger

    Set up a cron trigger to run daily at 9:00 AM, just like the countdown bot.

  2. 2

    Code Node (Calculate Day)

    Calculate days until December 25th AND the current day number (1-25) to match your spreadsheet.

  3. 3

    Google Sheets Node

    Read all rows from your prepared spreadsheet containing the jokes and facts.

  4. 4

    Code Node (Filter)

    Filter for today's day number. If multiple entries exist for the same day, pick one randomly.

  5. 5

    Gmail Node

    Send the joke or fact with the countdown included. Make it festive!
